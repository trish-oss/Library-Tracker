# Library-Tracker
A Python-based Library Book Tracker that uses JSON for data storage, allowing users to add, view, search, borrow, return, and delete books through a command-line interface.
Library Book Tracker (Python Project)



Features Add new books View all books Search books by title Borrow books Return books Delete books Data stored in JSON file (persistent storage)

Technologies Used

Python
JSON (for data storage)
File Handling
